免sudo使用docker

<!--more-->

### 1. 概述

每次在使用docker时都追加sudo会很麻烦，所以通过用户组使用户免sudo使用docker

### 2. 实现

* docker group

    ```shell
    sudo groupadd docker
    ```

* 将用户加入该 group 内

    ```shell
    sudo gpasswd -a ${USER} docker
    ```

* 重启 docker 服务

    ```shell
    sudo service docker restart
    ```

* 切换当前会话到新 group 或者重启 X 会话

    ```shell
    newgrp - docker
    ```
