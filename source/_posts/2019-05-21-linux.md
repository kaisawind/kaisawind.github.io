ubuntu禁用虚拟内存swap
<!--more-->

### 临时禁用

```shell
sudo swapoff -a
```

### 永久禁用

在swap分区这行前加 # 禁用掉，保存退出

```shell
sudo vim /etc/fstab
```

重新启动电脑

```shell
sudo reboot
```
