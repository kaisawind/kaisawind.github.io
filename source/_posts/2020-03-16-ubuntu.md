使用fwupd进行ubuntu固件更新
<!--more-->

## 1. 概述

打开ubuntu商店会报错「无法从fwupd下载固件」，所以需要手动进行固件的更新。

## 2. 更新方法

* Step1
```bash
sudo apt update && sudo apt upgrade -y
```

* Step2
```bash
sudo systemctl start fwupd
```

* Step3
```bash
sudo fwupdmgr refresh
```
![](/images/深度截图_选择区域_20200316102108.png)

* Step4
```bash
sudo fwupdmgr update
```
