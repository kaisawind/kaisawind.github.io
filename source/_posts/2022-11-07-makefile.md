Makefile中`:=`, `=`, `?=`, `+=`的区别

## `:=`

```makefile
A := A
B := $(A)B
A := AA
```

B的值为`AB`

## `=`

```makefile
A = A
B = $(A)B
A = AA
```

B的值为`AAB`

## `?=`

```makefile
A = A
A ?= AA
```

A的值为`A`


```makefile
A ?= AA
```

A的值为`AA`

## `+=`

```makefile
A = A
A += AA
```

A的值为`AAA`
