linux 图片批量压缩
<!--more-->

## 1. JPG

```bash
sudo apt-get install jpegoptim
```

通用方法
```bash
jpegoptim *.jpg
```

递归查找
```bash
find . -name '*.jpg' | xargs jpegoptim --strip-all
```
或
```bash
find -type f -name "*.jpg" -exec jpegoptim --strip-all {} \;
```

## 2. PNG

```bash
sudo apt-get install optipng
```

通用方法
```bash
optipng *.png
```

递归查找
```bash
find -type f -name "*.png" -exec optipng {} \;
```
