typora导出pdf时table可以占两页
<!--more-->

## 1. 概述

默认typora导出pdf时，当table一页不足时，会留空，然后换页输出，效果很不好。

## 2. CSS控制输出

在typora主题文件夹下配置自定义CSS
`~/.config/Typora/themes/base.user.css`

```css
@media print {
  table {
   page-break-inside: auto;
  }
}
```
