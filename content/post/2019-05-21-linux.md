ubuntu禁用虚拟内存swap
<!--more-->

> **提示**: Linux发行版更新较快，命令可能因版本不同而有差异。


### 临时禁用

```bash
sudo swapoff -a
```

### 永久禁用

在swap分区这行前加 # 禁用掉，保存退出

```bash
sudo vim /etc/fstab
```
或
```bash
sed -ri 's/.*swap.*/#&/' /etc/fstab
```

重新启动电脑

```bash
sudo reboot
```
