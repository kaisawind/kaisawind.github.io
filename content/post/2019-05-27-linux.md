linux查看外网IP
<!--more-->

> **提示**: Linux发行版更新较快，命令可能因版本不同而有差异。


* 1. ifconfig

    ```bash
    curl ifconfig.me
    ```

    结果
    ```bash
    114.114.114.114
    ```

* 2.  查IP(www.cip.cc)网站
    查IP(www.cip.cc)网站, 提供免费的IP查询服务,命令行查询IP, 并且支持'PC网站, 手机网站, 命令行(Windows/UNIX/Linux)' 三大平台, 是个多平台的IP查询网站

    ```bash
    curl cip.cc
    ```

    结果
    ```bash
    [master@master ~]$ curl cip.cc
    IP      : 114.114.114.114
    地址    : 中国  辽宁  大连
    运营商  : 移动

    数据二  : 辽宁省 | 移动

    数据三  : 中国辽宁省大连市 | 移动

    URL     : http://www.cip.cc/114.114.114.114
    ```
