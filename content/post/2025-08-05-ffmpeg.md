ffmpeg单mp4文件循环推流
<!--more-->

推流
```bash
ffmpeg -re -stream_loop -1 -i masthead.mp4 -vcodec h264 -acodec aac -f rtp_mpegts rtp://127.0.0.1:10000
```

播流
```bash
ffplay rtp://127.0.0.1:10000
```
