build linux kernel 4.4.194
<!--more-->

> **提示**: Linux发行版更新较快，命令可能因版本不同而有差异。


```bash
wget https://cdn.kernel.org/pub/linux/kernel/v4.x/linux-4.4.194.tar.gz
tar -xvf linux-4.4.194.tar.gz
cd linux-4.4.194
make mrproper
sudo apt-get install libncurses5-dev
make menuconfig
make -j4
make install
```
