## 1. 概述

mongodb中数据库记录的时间是int64型的时间戳，在进行聚合操作时需要把时间戳转换为时间。

## 2. 表达式

```
db.getCollection('alert_history').aggregate(
   [
     {
       '$project':
          {
            '_id': 1,
            'triggertime': 1,
            'time': { '$toDate': {'$multiply': ['$triggertime', 1000]} },
          }
     }
   ]
)
```
