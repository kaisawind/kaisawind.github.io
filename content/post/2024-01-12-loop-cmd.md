## 问题描述

需要在Linux中多次执行相同的命令。

## 解决方法

### 方法1：使用for循环

```bash
for i in {1..10}; do
    ls
done
```

### 方法2：使用seq

```bash
seq 10 | xargs -n 1 ls
```

### 方法3：使用find + exec

```bash
find . -name "*.txt" -exec ls {} \;
```

### 方法4：使用parallel

```bash
# 安装parallel
sudo apt-get install parallel

# 并行执行
parallel ls ::: dir/*.txt
```

### 方法5：使用xargs

```bash
# 生成命令
for i in {1..100}; do echo "ls"; done > commands.txt

# 并行执行
xargs -P 4 < commands.txt
```

## 性能对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| for循环 | 简单 | 顺序执行 |
| seq | 简洁 | 需要生成序列 |
| find + exec | 功能强大 | 对特殊文件名需要转义 |
| parallel | 真正并行 | 需要额外安装 |
| xargs | 并行可控 | 需要生成命令文件 |

## 最佳实践

1. **小量任务**：使用for循环即可
2. **大量任务**：使用parallel或xargs
3. **文件操作**：使用find + exec
4. **监控进度**：使用`echo`或`pv`显示进度
