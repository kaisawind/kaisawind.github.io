CentOS7安装k8s1.16.3

> **注意**: Kubernetes 1.16.3 已经是非常旧的版本（2019年发布），建议使用更新的版本。本文档保留作为历史参考。最新版本安装请参考[官方文档](https://kubernetes.io/docs/setup/production-environment/tools/kubeadm/install-kubeadm/)。
<!--more-->

## 前置要求

- CentOS 7
- 2GB或更多RAM
- 2个CPU或更多
- 集群中所有机器的网络连接
- 每个节点的唯一hostname、MAC地址和product_uuid

## 安装Docker

```bash
# 安装必要的依赖
yum install -y yum-utils device-mapper-persistent-data lvm2

# 添加Docker仓库
yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

# 安装Docker
yum install -y docker-ce docker-ce-cli containerd.io

# 启动Docker
systemctl start docker
systemctl enable docker
```

## 安装kubeadm、kubelet和kubectl

```bash
# 配置Kubernetes仓库
cat <<EOF > /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://packages.cloud.google.com/yum/repos/kubernetes-el7-x86_64
enabled=1
gpgcheck=1
repo_gpgcheck=1
gpgkey=https://packages.cloud.google.com/yum/doc/yum-key.gpg https://packages.cloud.google.com/yum/doc/rpm-package-key.gpg
EOF

# 安装指定版本
yum install -y kubeadm-1.16.3 kubelet-1.16.3 kubectl-1.16.3

# 启动kubelet
systemctl enable kubelet && systemctl start kubelet
```

## 初始化Master节点

```bash
kubeadm init --apiserver-advertise-address=192.168.1.118 --pod-network-cidr=10.244.0.0/16 --image-repository registry.cn-hangzhou.aliyuncs.com/google_containers
```

## 配置kubectl

```bash
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

## 安装网络插件

```bash
kubectl apply -f https://docs.projectcalico.org/v3.8/manifests/canal.yaml
```

## 加入Worker节点

```bash
kubeadm join 192.168.1.118:6443 --token huu57s.pnaqkyd5yodzbqhm \
    --discovery-token-ca-cert-hash sha256:0305678150fd8cc8e534bee7888849ba0e86fedce9e8d15d952e87fa8568a500
```

## 最新版本安装建议

对于生产环境，建议使用最新稳定版本。主要变化：

1. **Kubernetes仓库已更新**：使用 `pkgs.k8s.io` 仓库
2. **容器运行时**：推荐使用 containerd 而非 Docker
3. **网络插件**：Calico版本已更新
4. **版本升级**：支持就地升级

详细信息请参考：
- [Kubernetes官方文档](https://kubernetes.io/docs/setup/)
- [kubeadm安装指南](https://kubernetes.io/docs/setup/production-environment/tools/kubeadm/)
- [版本支持政策](https://kubernetes.io/docs/setup/release/version-skew-policy/)
